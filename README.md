# Linktree Alternative Prototype

計画書をもとにした、DB / 課金 / AI API 接続なしの静的プロトタイプです。

## 開き方

`index.html` をブラウザで開くと動きます。

GitHub Pages では、`main` ブランチのルートから公開します。

公開URL: https://ta713ku-ops.github.io/Linktree/

URLを共有して他の人に触ってもらう場合は、[URL共有用説明資料.md](./URL共有用説明資料.md) を使ってください。

## 入っているもの

- 21種類のテンプレート
- プラン切り替え UI
- プロフィール編集
- リンク編集
- セクション編集
- デザイン編集
- AIデザイン提案のダミー体験
- キープデザイン
- 分析画面のダミーデータ
- 公開設定 / QR / OGP / 独自ドメイン UI
- localStorage 保存
- 高解像度PNGベースの15テンプレート背景
- CSS構造ベースの6テンプレート（ヒーローカバー、特大カード、Bento、雑誌目次、SNSハブ、イベント告知）
- プロフィール画像 / ページ背景 / リンク背景 / リンク内画像のブラウザ内保存

## 注意

この段階では `node` / `npm` なしで動かせる検証版です。Vite + React + TypeScript + Tailwind へ移す場合も、データ構造と画面分割をそのまま移植しやすいようにしています。

画像カスタムは選んだブラウザの `localStorage` に保存されます。サーバーへアップロードしないため、別のブラウザや別の端末には共有されません。
